# Save a PhyloPic image

`save_phylopic` is a convenient function for saving a PhyloPic
silhouette fetched via
[`get_phylopic`](https://rphylopic.palaeoverse.org/reference/get_phylopic.md).
Available formats for saving are those available in the
[grDevices](https://rdrr.io/r/grDevices/grDevices-package.html) package
(i.e. pdf, png, svg, tiff, jpeg, and bmp).

## Usage

``` r
save_phylopic(img = NULL, path = NULL, bg = "transparent", ...)
```

## Arguments

- img:

  `picture`. A
  [Picture](https://rdrr.io/pkg/grImport2/man/Picture-class.html)
  object. For PhyloPic silhouettes, this is generated by
  [`get_phylopic`](https://rphylopic.palaeoverse.org/reference/get_phylopic.md)
  and
  [`pick_phylopic`](https://rphylopic.palaeoverse.org/reference/pick_phylopic.md).

- path:

  `character`. The output file path. The name of the file and file
  extension should be included (e.g. "./phylopic.png"). If this argument
  is `NULL` (default), the file is automatically saved in the current
  working directory as a png file.

- bg:

  `character`. The desired background color. By default this argument is
  set to "transparent". Note: .jpeg files are always exported with a
  background colour.

- ...:

  Additional arguments available for the called grDevices function. The
  called function is determined by the file extension used in `path`
  (e.g. [`png`](https://rdrr.io/r/grDevices/png.html)).

## Value

An image is saved. Nothing is returned to the R interface.

## Details

This function can be used to save a PhyloPic image obtained via the
[`get_phylopic`](https://rphylopic.palaeoverse.org/reference/get_phylopic.md)
or any object of class `Picture` from the grImport2 package. Any file
format available in the
[grDevices](https://rdrr.io/r/grDevices/grDevices-package.html) package
can be used (i.e. pdf, png, svg, tiff, jpeg, and bmp).

## Examples

``` r
if (FALSE) { # \dontrun{
# Get image data
img <- get_phylopic("27356f15-3cf8-47e8-ab41-71c6260b2724")
# Save file
save_phylopic(img = img, path = "./phylopic.png",
              width = 500, height = 500)
} # }
```
